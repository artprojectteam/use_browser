# Use Browser Information

## ブラウザで読み込む場合

**HTMLの記述**
```html
<script src="/path/to/use_browser.js"></script>
```

**JavaScriptの記述**
```javascript
var ub = window.UseBrowser;
// or
var ub2 = UseBrowser;
```

## npmインストール

```
npm i -D use-browser
```

**import**
```javascript
import UseBrowser from 'use-browser'
```
